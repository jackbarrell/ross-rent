# Assumptions and Limitations

## Assumptions

- STR market quality can be approximated using local comparables.
- Revenue model:
  - Monthly gross = ADR × occupancy × days/month
  - Annual gross = monthly × 12
- Operating costs include both variable and fixed cost buckets.
- Yield proxy = NOI ÷ purchase price.
- Scoring is a heuristic intended for screening, not underwriting.

## Limitations

- MLS and OTA data are mocked for PoC reliability.
- No permit/zoning compliance checks.
- No debt financing model (cash purchase implied in base metrics).
- Seasonality is represented by a single index per location.
- No historical time-series backtesting.
- No user auth or multi-user collaboration.
